Top Secrets

A web API for sharing secrets among authenticated users.

Routes

Route Request Payload Description
POST /api/v1/users { email, password } Registers a new user.
POST /api/v1/users/sessions { email, password } Log in as user.
DELETE /api/v1/users/sessions none Log out. If there is no user session, does nothing.
GET /api/v1/secrets none Get a list of all secrets being shared. Only available when logged in.
POST /api/v1/secrets { title, description } Create a new secret to share with other users. Only available when logged in.
